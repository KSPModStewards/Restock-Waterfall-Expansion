==============================================
Restock Waterfall Effects 0.3.0 by AtomicTech
==============================================

Restock Waterfall Effects contains a set of template and engine configuration files, that gives Knight of St. John's awesome waterfall effects to the stock engines.

This is a configuration mod for Restock and Waterfall and is built from Knight of St. John's Stock Waterfall Effects (0.6.3). You need to install the dependency mods for it to work.

================
ENGINES INCLUDED
================
- The Bobcat with a new Hydrolox Plume, along with the original SWE Hypergolic Plume.
- The Cheetah with a new Hypergolic Plume, along with the original SWE Hydrolox Plume.
- The Thud with its original Hypergolic Plume.
- The Reliant and the Swivel with their original SWE plumes with a Hypergolic Plume.
- The Goliath, Juno, Panther, Wheesley, Whiplash, and RAPIER's SWE Plumes all included.

============
DEPENDENCIES
============

Required:
- ModuleManager
- Waterfall
- B9PartSwitch

============
INSTALLATION
============

To install, place the GameData folder inside your Kerbal Space Program folder. If asked to overwrite files, please do so.
